# 实用工具箱 — 使用手册

---

## 一、打开工具箱

1. 找到 `toolbox.html` 文件
2. 双击打开（会自动用浏览器打开）
3. 如果没有自动打开，右键点击文件 → 选择"打开方式" → 选择 Chrome 或 Edge 浏览器

> 💡 不需要联网，不需要安装任何软件，打开就能用。

---

## 二、功能总览

打开后，页面顶部有 **8 个标签页**，点击即可切换功能：

| 标签 | 用途 |
|------|------|
| 批量创建文件夹 | 一次性创建很多文件夹 |
| 批量创建文件 | 一次性创建很多空文件 |
| 批量重命名文件 | 批量给文件改名 |
| 批量重命名文件夹 | 批量给文件夹改名 |
| 批量修改扩展名 | 统一修改文件后缀（如 .jpeg 改成 .jpg） |
| 文本对比 | 对比两段文字的不同之处 |
| 文本去重 | 去掉重复的行 |
| 文本排序 | 把文字按顺序排列 |

---

## 三、各功能详细操作

### 功能 1：批量创建文件夹

**场景**：需要一次性创建"1月"、"2月"、"3月"……等多个文件夹。

**操作步骤**：

1. 点击顶部"批量创建文件夹"标签
2. 在文本框中输入文件夹名称，**每行一个**：
   ```
   1月
   2月
   3月
   ```
3. 也可以从 Excel 表格中直接复制粘贴，工具会自动拆分
4. （可选）在"目标路径"中填写要创建文件夹的位置，例如 `D:\工作文件`。不填则需要把脚本放到目标位置再执行
5. 下方预览区会自动显示处理后的列表（已去重、去空行）
6. 点击 **"生成脚本"** 按钮
7. 浏览器会下载一个 `.bat` 文件（如 `批量创建文件夹_20260416.bat`）
8. 双击运行这个 `.bat` 文件，文件夹就创建好了
9. 窗口会显示"操作完成！"，按任意键关闭

---

### 功能 2：批量创建文件

**场景**：需要一次性创建多个空文件。

**操作步骤**：

1. 点击"批量创建文件"标签
2. 输入文件名，**每行一个，要带后缀名**：
   ```
   报告.docx
   数据表.xlsx
   说明.txt
   ```
3. （可选）填写目标路径
4. 点击"生成脚本" → 下载 `.bat` 文件 → 双击执行

> ⚠️ 如果文件已存在，脚本不会覆盖，可以放心执行。

---

### 功能 3：批量重命名文件

**场景**：需要给一批文件统一改名。

提供 **4 种改名方式**，点击对应的圆形按钮切换：

#### 方式一：直接映射

适合从 Excel 粘贴"旧名→新名"对照表。

1. 在 Excel 中准备两列：A 列是旧文件名，B 列是新文件名
2. 选中这两列，复制
3. 粘贴到文本框中（会自动识别 Tab 分隔）
4. 预览区会显示改名前后的对比
5. 确认无误后点击"生成脚本"

#### 方式二：查找替换

适合统一替换文件名中的某段文字。

1. 在"查找"框输入要替换的文字，如 `v1`
2. 在"替换为"框输入新文字，如 `v2`
3. 在下方文本框输入文件名列表
4. 预览区实时显示替换结果

#### 方式三：添加前缀/后缀

适合给所有文件名统一加前缀或后缀。

1. 在"前缀"框输入要添加到开头的文字，如 `2026年_`
2. 在"后缀"框输入要添加的后缀，如 `_终版`（会自动插入到扩展名之前）
3. 在下方文本框输入文件名列表

**举例**：文件名 `报告.docx`，前缀 `2026年_`，后缀 `_终版` → 结果 `2026年_报告_终版.docx`

#### 方式四：按序号编号

适合给文件按数字编号。

1. 设置起始号（默认 1）、步长（默认 1）、位数（默认 3 位，如 001、002）
2. 在下方文本框输入文件名列表
3. 文件名会被替换为序号，扩展名保留

**举例**：`照片A.jpg`、`照片B.jpg` → `001.jpg`、`002.jpg`

---

### 功能 4：批量重命名文件夹

操作方式与"批量重命名文件"完全相同，只是针对文件夹。

唯一区别：添加后缀时直接加在名称末尾（文件夹没有扩展名）。

---

### 功能 5：批量修改扩展名

**场景**：把一批 `.jpeg` 文件统一改成 `.jpg`。

**操作步骤**：

1. 点击"批量修改扩展名"标签
2. 在"原扩展名"框输入 `.jpeg`（前面的点可以不输，会自动补上）
3. 在"新扩展名"框输入 `.jpg`
4. （可选）填写目标路径
5. 预览区显示 `*.jpeg → *.jpg`
6. 点击"生成脚本" → 下载 → 双击执行

---

### 功能 6：文本对比

**场景**：对比两个版本的文字，找出修改了哪些内容。

**操作步骤**：

1. 点击"文本对比"标签
2. 在左边文本框粘贴**原始文本**
3. 在右边文本框粘贴**修改后的文本**
4. 点击 **"对比"** 按钮
5. 下方会显示对比结果：
   - 🟢 **绿色** = 新增的行
   - 🔴 **红色** = 删除的行
   - 🟡 **黄色** = 修改的行（具体改了哪些字会加深高亮）
6. 勾选"仅显示差异行"可以隐藏相同的行，只看不同之处

---

### 功能 7：文本去重

**场景**：一份名单里有重复的名字，需要去掉重复项。

**操作步骤**：

1. 点击"文本去重"标签
2. 在文本框中粘贴文本（每行一条）
3. 如果需要忽略大小写（比如 `ABC` 和 `abc` 算同一个），勾选"忽略大小写"
4. 点击 **"去重"** 按钮
5. 下方显示去重结果和统计信息（如"原始 100 行 → 去重后 75 行"）
6. 点击 **"一键复制"** 把结果复制到剪贴板

---

### 功能 8：文本排序

**场景**：把一份乱序的列表按顺序排好。

**操作步骤**：

1. 点击"文本排序"标签
2. 在文本框中粘贴文本
3. 选择排序方式：
   - **正序**：A→Z / 小→大
   - **倒序**：Z→A / 大→小
4. 选择排序模式：
   - **按字母排序**：按文字顺序排（支持中文拼音排序）
   - **按数字排序**：按数字大小排（非数字的行会排在最后）
5. 点击 **"排序"** 按钮
6. 点击 **"一键复制"** 把结果复制到剪贴板

## 四、常见问题

**Q：双击 .bat 文件后窗口一闪就没了？**
A：不会的，脚本执行完会显示"操作完成！"并等待你按键。如果确实一闪而过，可能是脚本有错误，请检查文件名是否包含特殊字符。

**Q：生成的脚本会覆盖已有的文件吗？**
A：不会。创建类脚本会自动检查文件/文件夹是否已存在，已存在的会跳过。

**Q：目标路径填错了怎么办？**
A：脚本会先检查路径是否存在，如果不存在会提示错误并停止，不会执行后续操作。

**Q：支持中文文件名吗？**
A：完全支持。脚本使用 UTF-8 编码，中文文件名不会乱码。

**Q：可以在手机上用吗？**
A：文本处理类功能（对比、去重、排序）可以在手机浏览器上使用。文件操作类功能生成的是 Windows .bat 脚本，只能在 Windows 电脑上执行。

---

## 五、清空与重置

每个功能面板都有 **"清空"** 按钮，点击后会重置该功能的所有输入和结果，方便开始新的操作。
