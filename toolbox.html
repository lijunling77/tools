📖 使用帮助

一、打开工具箱

双击 toolbox.html 文件即可用浏览器打开（推荐 Chrome 或 Edge）。不需要联网，不需要安装任何软件。

二、功能总览

标签用途
批量创建文件夹一次性创建很多文件夹
批量创建文件一次性创建很多空文件
批量重命名文件批量给文件改名
批量重命名文件夹批量给文件夹改名
批量修改扩展名统一修改文件后缀（如 .jpeg → .jpg）
文本对比对比两段文字的不同之处
文本去重去掉重复的行
文本排序把文字按顺序排列

三、文件操作类功能使用流程

  1. 在文本框中输入名称列表（每行一个），支持从 Excel 直接粘贴
  2. 可选：填写"目标路径"指定操作目录（如 D:\工作文件），留空则在脚本所在目录执行
  3. 工具会自动去重、去空行，预览区显示处理结果
  4. 点击"生成脚本"，浏览器会下载一个 .bat 文件
  5. 双击运行 .bat 文件，操作完成后窗口会显示"操作完成！"

四、批量重命名的 4 种模式

直接映射：从 Excel 粘贴两列数据（A 列旧名，B 列新名），自动识别。

查找替换：输入查找文本和替换文本，对所有文件名批量替换。

添加前缀/后缀：统一添加前缀或后缀。文件模式下后缀会插入到扩展名之前。
例：报告.docx + 前缀 2026年_ + 后缀 _终版 → 2026年_报告_终版.docx

按序号编号：用递增序号替换文件名，可设置起始号、步长和位数。
例：照片A.jpg、照片B.jpg → 001.jpg、002.jpg

五、文本对比颜色说明

  • 绿色 = 新增的行
  • 红色 = 删除的行
  • 黄色 = 修改的行（具体改了哪些字会加深高亮）

勾选"仅显示差异行"可以隐藏相同的行。

六、常见问题

Q：生成的脚本会覆盖已有的文件吗？
A：不会。创建类脚本会自动检查是否已存在，已存在的会跳过。

Q：目标路径填错了怎么办？
A：脚本会先检查路径是否存在，不存在会提示错误并停止。

Q：支持中文文件名吗？
A：完全支持，不会乱码。

Q：每个功能都有"清空"按钮，点击后会重置所有输入和结果，方便开始新的操作。
