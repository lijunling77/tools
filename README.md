# 🧰 实用工具箱

一个零门槛的本地小工具集，双击浏览器打开即用，无需安装任何软件。

## 特点

- **单文件** — 只有一个 `toolbox.html`，双击用浏览器打开就能用
- **零依赖** — 不需要网络，不需要安装任何东西
- **纯本地** — 所有操作在浏览器内完成，数据不会上传
- **中文界面** — 面向中文用户设计，操作简单直观

## 包含 8 个功能

### 文件操作类（生成 .bat 脚本）

| 功能 | 说明 |
|------|------|
| **批量创建文件夹** | 输入文件夹名称列表，生成一键创建脚本 |
| **批量创建文件** | 输入文件名列表（含扩展名），生成一键创建空文件脚本 |
| **批量重命名文件** | 支持直接映射、查找替换、添加前缀/后缀、按序号编号 |
| **批量重命名文件夹** | 同上，针对文件夹 |
| **批量修改扩展名** | 输入原扩展名和新扩展名，一键转换 |

### 文本处理类（浏览器内完成）

| 功能 | 说明 |
|------|------|
| **文本对比** | 逐行对比两段文本，高亮显示差异（支持字符级差异） |
| **文本去重** | 去除重复行，支持忽略大小写 |
| **文本排序** | 按行排序，支持正序/倒序、字母/数字排序 |

## 使用方法

### 快速开始

1. 下载 `toolbox.html` 文件
2. 双击用浏览器打开（推荐 Chrome / Edge）
3. 点击顶部标签页切换功能，按提示操作即可

### 文件操作类功能的使用流程

1. 在文本框中输入名称列表（每行一个），支持从 Excel 直接粘贴
2. **可选**：填写"目标路径"指定操作目录（如 `D:\工作文件`），留空则在脚本所在目录执行
3. 工具会自动去重、去空行、去首尾空格，并在预览区显示处理结果
4. 点击"生成脚本"按钮，浏览器会下载一个 `.bat` 文件
5. 将下载的 `.bat` 文件放到目标文件夹（如果没填目标路径的话），双击执行

### 批量重命名的 4 种模式

- **直接映射**：每行输入 `原名[Tab]新名`，支持从 Excel 粘贴两列数据
- **查找替换**：输入查找文本和替换文本，对所有文件名批量替换
- **添加前缀/后缀**：为所有文件名统一添加前缀或后缀（文件模式下后缀插入到扩展名之前）
- **按序号编号**：用递增序号替换文件名，可设置起始号、步长和位数

### 文本处理类功能

- 在文本框中粘贴文本，点击操作按钮即可
- 结果可通过"一键复制"按钮复制到剪贴板

## 生成的脚本说明

- 脚本使用 UTF-8 编码，支持中文文件名
- 创建类脚本会自动检查文件/文件夹是否已存在，避免覆盖
- 如果指定了目标路径，脚本会先检查路径是否存在
- 执行完成后会显示"操作完成！"并暂停，方便确认结果
- 下载的文件名格式为 `功能名称_日期.bat`（如 `批量创建文件夹_20260416.bat`）
